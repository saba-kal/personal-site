TIC-TAC-TOE AI

Below is a tic-tac-toe game implemented in TypeScript. The AI for this game uses the minimax algorithm, largely based on a guide found here. It is not a very elegant solution, as it brute forces its way to victory by checking all possible outcomes. However, it is perfectly feasible for a simple game like Tic-Tac-Toe.
