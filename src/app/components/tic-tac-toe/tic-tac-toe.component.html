Info

The AI for this game uses the minimax algorithm. The implementation of it on this website is largely based on a guide found here. It is not a very elegant solution, as it pretty much brute forces its way to victory by checking all possible outcomes. However, it is perfectly feasible for a simple game like Tic-Tac-Toe.
